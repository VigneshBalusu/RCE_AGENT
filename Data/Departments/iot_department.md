## 2. Vision & Mission (IoT Specialization)
* **Search Tags:** **IOT_VISION, IOT_MISSION, IOT_GOALS.**

### Vision
"To develop competent professionals in the field of **Internet of Things (IoT)** with a strong foundation in hardware-software integration, data analytics, and sensor technology to solve societal problems."

### Mission
1.  **Technical Expertise:** To provide in-depth knowledge of **Embedded Systems, Sensors, and Cloud Computing**.
2.  **Innovation:** To encourage the development of smart solutions for **Smart Cities, Agriculture, and Healthcare**.
3.  **Industry Readiness:** To train students in industry-standard tools like **Raspberry Pi, Arduino, and AWS IoT**.

## 4. Academic Focus & Curriculum
* **Search Tags:** **IOT_SYLLABUS, IOT_SUBJECTS, IOT_CURRICULUM.**
* **Core Subjects:**
    * **IoT Architecture:** Perception, Network, and Application layers.
    * **Embedded Systems:** Microcontrollers (8051, ARM), RTOS.
    * **Sensor Technology:** Temperature, Proximity, Gas, and Motion sensors.
    * **Communication Protocols:** MQTT, CoAP, Zigbee, Bluetooth BLE, LoRaWAN.
    * **Cloud & Data:** AWS IoT Core, Python for Data Science, Big Data Analytics.

## 5. Laboratories & Infrastructure
* **Search Tags:** **IOT_LABS, SENSORS_LAB, ARDUINO_LAB, EMBEDDED_IOT_LAB.**
* **Overview:** The specialization features dedicated labs bridging the gap between **Hardware (Electronics)** and **Software (Coding)**.

### Key Laboratories
1.  **IoT Sensors & Devices Lab:**
    * **Equipment:** Arduino Uno, NodeMCU (ESP8266), Raspberry Pi 4 kits.
    * **Focus:** Interfacing sensors (DHT11, Ultrasonic, IR) with cloud platforms.
2.  **Embedded Systems Lab:**
    * **Equipment:** ARM Cortex Boards, Keil ÂµVision IDE.
    * **Focus:** Programming microcontrollers for real-time applications.
3.  **Advanced Python & Data Science Lab:**
    * **Focus:** Analyzing sensor data using Python libraries (Pandas, NumPy, Matplotlib).
4.  **Cloud Computing Lab:**
    * **Focus:** Connecting physical devices to AWS/Azure for remote monitoring and control.
5.  **Innovation/Project Lab:**
    * **Activity:** Building prototypes like "Smart Home Automation", "Smart Irrigation Systems", and "Health Monitoring Bands".
